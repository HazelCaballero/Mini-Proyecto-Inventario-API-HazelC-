Frontend - Inventario API

Este frontend permite interactuar con la API REST de inventario desarrollada en Django y Django REST Framework. Permite crear, listar, actualizar y eliminar productos de forma visual y sencilla.


1. Instalación de dependencias

El frontend está desarrollado en **React**.

a)Descarga e instala Node.js desde [nodejs.org](https://nodejs.org/).

b) Instalar dependencias del proyecto

Abre la terminal en la carpeta `front` y ejecuta:

npm install



2. Ejecución del frontend

En la terminal, ejecuta:

npm start


3. Herramientas y razones de uso

React: Permite construir interfaces de usuario de forma eficiente y modular.
Fetch: Para hacer solicitudes HTTP a la API Django.


4. Funcionalidades implementadas

Listar productos: Muestra todos los productos disponibles en la base de datos.
Crear producto: Permite agregar un nuevo producto mediante un formulario.
Editar producto: Permite modificar los datos de un producto existente.
Eliminar producto: Permite borrar un producto del inventario.
